# Riju M — Personal Portfolio Website

A single-page personal introduction website built with plain HTML, CSS, and JavaScript. No frameworks, no build tools — just open `index.html` and it works.

🔗 **Live site:** https://rijum3107-cmd.github.io/Portfolio-/

## About

This is a personal intro page created as a college project to showcase who I am, what I'm learning, and how to get in touch. It includes:

- A hero section with an animated background and intro
- An About section with a short bio and quick facts
- A Skills section listing languages and tools I'm learning
- An Interests section (sports, animation, game development)
- A Contact section with email and GitHub links

## Built With

- **HTML5** — page structure
- **CSS3** — styling, animations, and responsive layout (no frameworks like Bootstrap)
- **Vanilla JavaScript** — scroll-reveal animations and active nav-link highlighting
- **Google Fonts** — Fraunces (headings), Sora (body text), JetBrains Mono (labels)

## Features

- Fully responsive (works on mobile, tablet, and desktop)
- Smooth scroll navigation
- Scroll-triggered fade-in animations
- Animated gradient background in the hero section
- Hover effects on cards and buttons
- Respects `prefers-reduced-motion` for accessibility

## Running Locally

No installation needed. Just download `index.html` and open it directly in any web browser.

## Deployment

This site is hosted for free using **GitHub Pages**:

1. Push the code to a GitHub repository
2. Go to **Settings → Pages**
3. Under "Build and deployment," set Source to `Deploy from a branch`
4. Select the `main` branch and `/ (root)` folder → Save
5. Your site goes live at `https://<username>.github.io/<repo-name>/`

## Customization

To make this your own, edit `index.html` and update:

- Name, college, and bio text in the hero and About section
- Skills/chips in the Skills section
- Interests in the "Beyond the Code" section
- Email and GitHub links in the Contact section (search for `mailto:` and `github.com`)
